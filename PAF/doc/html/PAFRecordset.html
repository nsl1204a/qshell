public abstract class PAFRecordset extends PAFObject

Clase que proporciona el interface a implementar en todos los Recordset.
Un recordset es una abstracción de un filtro sobre una fuente de datos determinada.Esto es, representa el conjunto de registros que se recuperan tras una consultacon unas condiciones determinadas sobre una fuente de datos cualquiera.
