public class PAFConfiguration extends PAFObject

Clase de configuración general.
Consiste en varias colección que el usuario deberá rellenar con el contenido deseado.Uno de los principales usos que tiene esta página es cargar en un solo objeto todos los datosnecesarios para l aejecución de una página.Las colecciones que contiene son:. Colección de DataSources.. Colección de Recordsets.. Colección de Variables.. Colección de Globales.
